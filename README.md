# Simple Weather App
Погодное приложение, которое отображает текущую погоду и прогноз на ближайшие 3 дня. Есть возможность сменить город.  
  
Данные берутся из сервиса https://www.accuweather.com/. Хождение в сеть через Retrofit2 + Kotlin coroutines.  
Для того чтобы собрать свою apk необходимо положить ключ приложения в local.properties: ```api.accuweather.key=YOUR_KEY```
